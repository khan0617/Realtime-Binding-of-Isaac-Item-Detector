scraping.scraper module

scraper.py contains utilities for collecting all Isaac item info we need from the Wiki’s HTML and downloading their images. Also provides a utility to dump / retrieve isaac item data from JSON.

scraping.scraper.download_item_images(isaac_items, data_dir, item_dir, parallel=True)[source]

Downloads images for all provided IsaacItems.

Parameters:

  • isaac_items (List[IsaacItem]) – A list of IsaacItems.

  • data_dir (str) – The root directory where image folders will be saved.

  • item_dir (str) – Subdirectory within data_dir for item images. Ex: If you want the individual item folders to be stored in ‘data/items/, pass in data_dir=’data’, and item_dir=’items’. This function handles concatenation.

  • parallel (bool) – If true, parallelize the downloads, else do them sequentially.

Return type:

None

scraping.scraper.get_isaac_items_from_json(filename)[source]

Attempt to parse IsaacItems from the provided json filename.

Isaac items should be stored in the following example format (item.get_image_id_tail() is the main key):

“396”: {

“name”: “Ventricle Razor”, “item_id”: “5.100.396”, “img_url”: “https://static.wikia.nocookie.net/bindingofisaacre_gamepedia/images/9/97/Collectible_Ventricle_Razor_icon.png/revision/latest?cb=20210821162403”, “wiki_url”: “https://bindingofisaacrebirth.fandom.com/wiki/Ventricle_Razor”, “description”: “Creates up to two portals that remain even if Isaac leaves the room. Upon entering a portal, Isaac is teleported to the other portal.”, “item_quality”: “1”, “quote”: “Short cutter”, “url_encoded_name”: “Ventricle_Razor”, “yolo_class_id”: “91”

},

Parameters:

filename (str) – The name of the json file with IsaacItem info.

Return type:

list[IsaacItem] | None

Returns:

A list of IsaacItems. Upon failure, return None.
